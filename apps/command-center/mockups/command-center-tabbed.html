Task Board Architecture

A file-based project orchestration system with cross-platform support.

Overview

Task Board is built with a local-first architecture that keeps all data in JSON files, syncing to GitHub for cross-device access. The system consists of two apps:

  • Command Center (Desktop) - Tauri 2.0 + React for deep work sessions
  • Launchpad (Mobile) - React Native for check-ins and voice instructions
